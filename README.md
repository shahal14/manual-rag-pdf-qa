\# Manual RAG for PDF Question Answering



This project implements a fully manual Retrieval-Augmented Generation (RAG) pipeline

for answering questions from PDF documents.



\## Features

\- PDF text ingestion

\- Chunking and semantic embeddings

\- FAISS-based retrieval

\- Extractive, citation-backed answers

\- Streamlit UI



\## Tech Stack

\- Python

\- Streamlit

\- FAISS

\- SentenceTransformers

\- PyMuPDF



\## How to Run

```bash

python -m streamlit run app/streamlit\_app.py
